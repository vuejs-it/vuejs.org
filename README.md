# vuejs.org

This site is built with [hexo](http://hexo.io/). Site content is written in Markdown format located in `source`. Pull requests welcome!

## Developing

Start a dev server at `localhost:4000`:

```
$ npm install -g hexo-cli
$ npm install
$ hexo server
```
